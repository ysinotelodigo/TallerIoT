Bienvenido a este taller de IoT

Tutoriales

Configuración

Este tutorial explicará los pasos que hay que seguir para configurar el entorno de desarrollo Arduino IDE para que pueda ser utilizado con la placa NodeMCU.

¡Hola Mundo!/Hello World!

Este tutorial explicará algunos conceptos básico de la programación al mismo tiempo que se logrará ejecutar nuestra primer programa para la placa NodeMCU.

Proyecto Led

El objetivo de este ejemplo es permitir un primer contacto con la placa NodeMCU, la protoboard y algunos componentes electrónicos.

Proyecto Calculadora

El objetivo de este tutorial es mostrar un ejemplo de comunicación bidireccional entre la placa y el ordenador por el cable de serie (cable usb). Para ello se desarrollará una calculadora.

Semáforo Básico

Este proyecto tiene como objetivo verificar el correcto funcionamiento de un circuito sencillo que simula un semáforo. Posteriormente en los dos siguientes proyectos será usado este circuito.

Semáforo Móvil

El objetivo de este ejemplo es mostrar cómo crear un punto de acceso Wifi con la placa NodeMCU y un mini-servidor de páginas web que proporcionará una página al dispositivo móvil que se conecte para poder manejar las luces del semáforo.

Telégrafo

El objetivo de este ejemplo es mostrar cómo se puede conectar la placa a un broker ("servidor") mediante el protocolo MQTT. Para ello se ha desarrollado un ejemplo de un telégrafo en el que dos personas se podrán comunicar mediante un botón y un zumbador.
